## 🎨 Painting Production Project
#### 사용자가 업로드한 그림과 그림판을 이용해서 실제 본인이 직접 그린 그림을 사물인식기능을 활용하여 유화로 변환해주는 사이트

## 목표
-   사물인식기능 활용
-   GitHub 활용 및 효율적인 분업
-   DRF를 활용하여 실력 높이기
